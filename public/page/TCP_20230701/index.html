鹿児島方言のアクセント体系
  1. 二型アクセント体系
    鹿児島方言は２つのアクセント型を持つ二型アクセント体系である。
    • A 型： 語末から 2 つ目の音節にアクセントが置かれる (HL#)
      e.g. アイ [サ] ツ (挨拶)
    • B 型： 語末音節にアクセントが置かれる (H#)
      e.g. ショウ [ユ (醤油)
  2. シラビーム方言
    鹿児島方言ではアクセント決定における計算は音節を単位として行なわれ、また、アクセントを担う単位も音節である。
    e.g. [ゼン] ゴ (前後) , ネ [ガイ (願い)
  3. 文節単位でのアクセント実現
    アクセントの実現ドメインは「文節」であり、助詞や助詞連続が原則として自らのアクセントを持たない。
    e.g. アイサ [ツ] ヲ (挨拶を) 、アイサツ [カ] ラ (挨拶から)
  4. 複合語アクセント
    鹿児島伝統方言の複合語は、その前部要素のアクセント型を継承する。これは「一般複合法則」もしくは「平山の法則」と呼ばれる (平山, 1951; 上野, 1997; 早田, 1999)。
    e.g. ア [オ (青) + エン [ピ] ツ (鉛筆) = アオエンピ [ツ (青鉛筆)
先行研究と分野の状況
  1. 鹿児島方言のアクセント変化における東京方言の影響
    • 鹿児島方言において現在進行中のアクセント変化は東京方言の干渉によるもの
      • 言語内/外のどちらの側面から見ても統一的に言える
        • 言語内: 「アクセント下降の有無」が東京方言に接近する方向で変化しているから
        • 言語外: 東京方言式に向かうアクセント変化」と「話者の都会的意識」および「マスメディアをとおした東京方言への接触の度合い」が相関しているから
  2. 鹿児島方言アクセントについての理論的研究
    • 鹿児島方言アクセントについての理論的研究は「伝統方言の単独語アクセント」を扱ったもののみであり、「アクセント変化」や「複合語アクセント」は未だ論じられていない。
